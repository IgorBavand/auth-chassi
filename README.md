# Auth Chassi

## Technologies:

>Java 17

> SpringBoot 3.0.2

> Spring Security 6

> Flyway 9.15.1

> Json Web Token

## Installation:

Maven installed is required (apt-get install maven)

1. `mvn install` for install dependencies
2. `mvn spring-boot:run` for run application
3. Acess the url: [http://localhost:8081](http://localhost:8081)
